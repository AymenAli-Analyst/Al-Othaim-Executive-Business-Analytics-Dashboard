# 📊 Al Othaim Retail Analytics Dashboard 
## 🚀 Project Overview

This project analyzes Al Othaim Markets retail performance across Saudi Arabia between 2019 and 2025 using Power BI.

The dashboard helps decision-makers monitor sales, profitability, customer behavior, branch performance, and category trends.

## 🔍 Key Insights

### 🏙️ Top Revenue City
🥇 Riyadh generated the highest revenue.

### 📦 Top Categories
- Bakery
- Beverages
- Dairy
- Personal Care

### 📉 Revenue Trend
Revenue peaked in 2022 and declined afterward.

### ⭐ Customer Experience
Customer ratings remain stable but show room for improvement.

### 🏪 Branch Performance
Performance varies significantly between branches.

---

## ⚠️ Business Problems

🔴 Revenue decline after 2022

🔴 Uneven branch profitability

🔴 Moderate customer satisfaction

🔴 Revenue concentration in major cities

🔴 Low-profit products consuming resources

- Benchmark top-performing branches
